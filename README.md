**Pre-requisites:**
- java (v1.8)
- gradle (v3.3)

## Code style (eclipse users only)
Import the file [code-style.xml](code-style.xml) into your IDE.

## Test
```bash
gradle clean test
```

## Build
```bash
gradle clean assemble
```

- Insertion Sort
